Create a Cyberpunk Personal Portfolio website.

**Visual Style:**
- **Theme:** Dark, gritty, neon-soaked cyberpunk aesthetic. High contrast.
- **Colors:** Deep blacks (#050505), neon pink (#ff00ff), cyan (#00ffff), and electric blue.
- **Typography:** Monospace fonts (like JetBrains Mono or Fira Code) for headers, clean sans-serif for body text. Glitch effects on hover.

**Key Sections:**
1.  **Hero Section:**
    - Large, glitching text: "WAKE UP, SAMURAI."
    - Background: Animated digital rain or a distorted cityscape loop.
    - CTA Button: "Enter the Matrix" (Neon border, glow effect).

2.  **About Me:**
    - "Netrunner Profile" style card.
    - Skill bars styled as system diagnostics (Load: 98%).
    - Avatar with a holographic overlay effect.

3.  **Projects (The Deck):**
    - Grid layout representing "Data Shards".
    - Hovering over a project card reveals details with a CRT monitor scanline effect.
    - Tags: React, Node, Web3, AI.

4.  **Contact (Uplink):**
    - A terminal-style input form.
    - Placeholder text: "Establish connection..."
    - Submit button says "Transmit Data".

**Tech Stack Requirements:**
- Next.js 14+ (App Router)
- Tailwind CSS
- Framer Motion for glitch/reveal animations
- Lucide React for icons
